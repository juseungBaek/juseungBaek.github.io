JAVA

Spring

SpringBoot

Spring Boot로 REST API를 구현하고 공공 데이터 API 및 Naver Maps API를 활용한. 경험이 있습니다. 스케줄러로 주기적으로 데이터를 쌓은 후 평균을 계산하여 프로젝트를 개선했습니다. 데이터 JPA와 JPQL을 활용하여 로그인, 카카오 로그인 및 Q&A 페이지를 구현했습니다.

HTML/CSS

HTML5으로 웹퍼블리싱이 가능하고, EL문과 JSTL 문법을 사용하여 JSP페이지에서 데이터를 원하는 위치에 표기할 수 있습니다. 부트스트랩을 활용해서 만든 뒤, 상세한 설정들은 CSS로 설정할 수 있습니다. 미디어 쿼리로 모바일 환경에서도 잘 표기될 수 있게 훈련하고 있습니다.

JavaScript
